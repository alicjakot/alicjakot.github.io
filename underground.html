The resilience of London Underground
Network

Overview
This project investigated the resilience of London’s underground network and its limitations using network analysis. It considered the infrastructural network as well as commuting flows. 

Methods
To estimate the importance of the nodes in London underground network I used degree, closeness and betweenness centralities. The study of London’s underground resilience followed two different strategies: non-sequential node removal, where ten nodes with highest centrality have been selected and removed, with new centrality being recomputed only after the removal of ten nodes; and sequential node removal, where the node with highest centrality was removed, the centrality measure was being recomputed and the new top-scoring node was removed. For sequential strategy ten top nodes have been removed. To evaluate the impact the node removal had on the London underground network I calculated average shortest path and diameter of the network.

This project was originally written as part of a master's degree in the Centre for Advanced Spatial Analysis, Bartlett Faculty of the Build Environment, UCL.
